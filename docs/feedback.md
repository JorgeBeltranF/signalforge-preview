# Feedback

The most useful preview feedback is practical:

- Did install work cleanly?
- How long did it take to reach the first useful insight?
- What part of the report felt most helpful?
- What part felt confusing or too internal?
- Did compare feel meaningfully different from a generic HTML export?
- If you use Dynatrace: did you notice the Evidence Relationships section in the report?
- Did the categories (Agreement, No Corroboration, Partial Agreement, Insufficient Evidence) make sense without reading the docs?
- Did the Evidence Relationships section change how you investigated the run?
- Did you trust the cross-source relationships shown, or did you still need to open Dynatrace to verify them?

Recommended issue categories:

- install-help
- preview-feedback
- bug
- question
